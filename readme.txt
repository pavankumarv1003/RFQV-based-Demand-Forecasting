
## How to Run the Project

1. Activate the virtual environment

   ```bash
   venv\Scripts\Activate
   ```

2. Open the project folder

   ```bash
   cd retail_forecaster
   ```

3. Run the server

   ```bash
   python manage.py runserver
   ```
